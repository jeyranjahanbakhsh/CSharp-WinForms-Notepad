# Simple Notepad (C# WinForms)

Simple Notepad is a lightweight text editor built with C# and Windows Forms,
inspired by the classic Windows Notepad application.

## Features
- Create, open, save, and save-as text files
- Detect unsaved changes with confirmation dialogs
- Display file name and modified state (*) in the window title
- Cut, copy, paste, and select all
- Word wrap support
- Keyboard shortcuts (Ctrl+N, Ctrl+O, Ctrl+S, etc.)
- About dialog

## Technologies
- C#
- Windows Forms
- System.IO

## Purpose
This project is designed as an educational example to practice:
- WinForms UI development
- File handling
- Event-driven programming
- User experience basics
